Структура репозитория

/data/img/ - содержит изображения отпечатков пальцев
/data/ref/ - содержит вручную сегментированные изображения отпечатков пальцев
/data/seg/ - содержит маску сегментации предлагаемым алгоритмом
/data/view/ - содержит маску сегментации предлагаемым алгоритмом наложенную на изображение

/examples_learn_pkl/ - содержит наборы данных для обучения нейронной сети
/examples_check_pkl/ - содержит наборы данных для проверки нейронной сети при обучении
/examples_valid_pkl/ - содержит наборы данных для итоговой проверки предлагаемого алгоритма

/models_json/ - содержит файлы моделей нейронных сетей
/models_hdf5/ - содержит файлы весов обученных нейронных сетей

/report_text/ - содержит результаты работы алгоритма с различными параметрами

/lib/ - содержит файлы исходных кодов работы алгоритма
/lib/create.py - файл создания наборов данных используемых при обучении нейронной сети
